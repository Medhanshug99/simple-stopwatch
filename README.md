#  Stopwatch Web App

A simple and responsive stopwatch   
It allows users to start, pause, and reset time with accurate millisecond tracking.

## Live Preview : 
(https://simple-stopwatch-js.vercel.app/)

##  Features
- Start / Pause functionality
- Reset stopwatch
- Millisecond precision
